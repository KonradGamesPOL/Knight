Install Knight game

Its a platformer game, with relaxing music and nice pixelart graphics. Its an exe file, and you have to download it form GitHub. To install it, scroll down the Github page, till you find "Assets". Then, install only the exe file. Open the game, and you can play!

Important: When you install the game from Github, there might appear "Dangerous File" warning. You have to make sure that your web browser installs the file properly.
