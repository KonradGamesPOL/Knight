Knight Game

Its a platformer game, with relaxing music and nice pixelart graphics. Its avaiable for Windows, Linux, MacOS and Android, so you can play even on your phone! To install it, scroll down the Github page, till you find the newest version. Open the game, and you can play!

It features:

- 4 unique levels,
- Hardcore level,
- Open World level, which is currently beeing developed,
- Stats (you can save them!)
- Many items, enemies, platforms, keys, coins, stars...
- And more for you to discover!


Important: When you install the game from Github, there might appear "Dangerous File" warning. You have to make sure that your web browser installs the file properly.


Have fun playing!
